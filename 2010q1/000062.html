Hallo,

hier eine Zusammenfassung, was sich zuletzt auf http://www.openthesaurus.de 
getan hat:

* Bei den Suchergebnissen ist jetzt unsere befreundetes Projekt 
http://45info.com verlinkt. Auf 45info.com kann man sich Zusammenfassungen 
von Wikipedia-Artikeln vorlesen lassen und die Bilder dazu anschauen.

* OpenThesaurus auf Android: im Market gibt es seit kurzem eine kostenfreie 
App, mit der man OpenThesaurus bequem von Android-Handys aus nutzen kann. 
Einfach im Android-Market nach "OpenThesaurus" suchen.

* Der Thesaurus für OpenOffice.org auf openthesaurus.de war leider eine Zeit 
lang fehlerhaft - es fehlten manche Wörter. Wer das *.oxt manuell 
installiert hat, sollte sich die aktuelle Version von 
http://www.openthesaurus.de/about/download herunterladen und installieren

* für Entwickler: die API wurde um eine Suche nach ähnlich geschriebenen 
Wörtern erweitert - siehe http://www.openthesaurus.de/about/api


Aktuellere Informationen als auf dieser Mailing-Liste bekommt man, wenn man  
http://twitter.com/openthesaurus folgt oder sich auf der ganz neu 
eingerichteten Facebook-Seite als Fan einträgt:
http://www.facebook.com/pages/OpenThesaurus/105692912797047
