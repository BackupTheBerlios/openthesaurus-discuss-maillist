OpenThesaurus (http://thesaurus.kdenews.org) - that's what it actually is
from - is GPL. The GPL says that you have to provide or offer the "source"
a.k.a. preferred form of modification.

Courrently, upstream only provides the thesaurus binary files for
dowload and a database dump. This database dump is not the source in
sense of the GPL sense because it _may_ be the preffered form of
modification but it does not contain the scripts which are needed to get
this dump into a form where thencode (which creates the thesaurus
binaries) can process it.

As I did ask for those scripts (a.k.a part of the source in the GPL
sense) I didn't get it. (No offense intended, this is just a report.)

So I could not package this even if I wanted to...

Either I get the "full source" or I have to wait for OOo 2.0 in which
the thesaurus will get another format (where it is actually editable
text...)
